# exile_roof_ridge_tiles

Adds a dedicated roof ridge tile for Exile builds.

## Purpose

This mod provides a single special roof piece used as a ridge/cap tile.

It is useful when:
- A roof has an odd total width and leaves a 1-node center gap between sloped tiles.
- You want a clean top cap over that central seam.
- You want to build narrow flat roof lines with a matching tile style.

## Included node

- `exile_roof_ridge_tiles:roof_ridge_tile`

## Recipe

- Craft type: `brick_makers_bench`
- Input: `tech:roof_tile`
- Output: `exile_roof_ridge_tiles:roof_ridge_tile`

## Notes

- This mod intentionally only adds the ridge/cap tile and its recipe.
- It depends on `tech` for base roof tile materials and `minimal` for stack settings.
